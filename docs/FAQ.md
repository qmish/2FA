# FAQ и глоссарий

## FAQ

**Q: Где найти OpenAPI спецификацию?**  
A: В `docs/openapi.yaml`.

**Q: Как быстро запустить проект локально?**  
A: Для K3D — `docs/k3d-deployment.md`, для локального запуска — раздел "Быстрый запуск".

**Q: Где настройки production?**  
A: В `docs/production.md` и чек‑лист `docs/PROD_CHECKLIST.md`.

**Q: Как получить доступ к UI?**  
A: См. `docs/UI_ACCESS.md`.

**Q: Какие метрики доступны?**  
A: См. `docs/observability.md`.

## Глоссарий

- **2FA** — двухфакторная аутентификация.
- **RADIUS** — протокол сетевой аутентификации (UDP/1812).
- **OTP** — одноразовый код.
- **Passkey / WebAuthn** — механизм пароль‑less аутентификации.
- **JWT** — токен доступа/обновления.
- **Ingress** — внешний вход в кластер Kubernetes.
- **HPA** — горизонтальное автоскалирование.
