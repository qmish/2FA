## Политика доступа к логам и метрикам

### Принципы
- Доступ только по ролям (read-only для большинства, admin для изменений).
- Запрет публичного доступа к Grafana/Prometheus/Loki.
- Доступ через VPN или корпоративный SSO.

### Роли
- **Viewer**: просмотр дашбордов/метрик.
- **Editor**: изменение дашбордов.
- **Admin**: управление источниками данных и пользователями.

### Рекомендации
- Включите RBAC в Grafana.
- Ограничьте доступ к Prometheus через Ingress/BasicAuth/SSO.
- Логи в Loki доступны только через Grafana.
