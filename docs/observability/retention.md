## Политика ретенции

### Метрики (Prometheus)
- Рекомендуемая ретенция: 15–30 дней (в зависимости от объёма и SLA).
- Пример флага: `--storage.tsdb.retention.time=30d`.
- Для долгого хранения используйте удалённое хранилище (Thanos/Cortex/Mimir).

### Логи (Loki)
- Рекомендуемая ретенция: 7–30 дней.
- Пример настройки: `limits_config.retention_period: 168h`.
- Для длительного хранения используйте объектное хранилище (S3/GCS/MinIO).

### Alertmanager
- Рекомендуемая ретенция: 5–15 дней.
- Контролируйте диск/volume для `/alertmanager`.
