# Политика доступа к админке и RADIUS

## Админка
- Доступ только по admin JWT.
- Эндпоинт входа `/api/v1/admin/auth/login` доступен без токена.
- Рекомендуется ограничить доступ по IP (Ingress/Firewall).
- Рекомендуется включить MFA для админов.

## RADIUS
- Ограничить IP диапазоны клиентов (`radius_clients.ip`).
- Использовать сильный `RADIUS_SECRET` и ротацию ключа.
- Ограничить доступ к UDP/1812 на уровне сети.

## Рекомендации
- Логи админских операций должны храниться (audit).
- Регулярно проверять список администраторов и клиентов RADIUS.
